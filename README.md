# Unga Bunga Caveman Template is a free REAPER mix template designed for modern metal production.

It includes pre-configured routing, mix buses, and a full session structure so producers can start mixing immediately without building their workflow from scratch.

The template is designed for modern metal, djent, and heavy guitar music and can also serve as a learning tool for beginners wanting to understand professional mix routing.
